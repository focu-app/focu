Access a powerful Pomodoro timer right from your menu bar. Simple, distraction-free, and always ready when you need to focus.

> The Pomodoro Technique helps you work with time, instead of against it. Break your work into focused 25-minute sessions.

## Key Features

### Timer Options
- Pomodoro: 25-minute focus sessions
- Short Break: 5-minute recharge
- Long Break: 15-minute rest
- One-click start/pause

### Menu Bar Integration
- Always accessible
- Minimal interface
- Current task display
- Session progress at a glance

### Task Display
- See your current focus task
- Quick task switching
- Expand to full Focus Page
- Track work sessions

## Using the Timer

### Quick Start
1. Click the menu bar icon
2. Select timer type
3. (Optional) Enter your task
4. Click Start

### Session Flow
1. Work focused for 25 minutes
2. Take a 5-minute break
3. After 4 sessions, take a 15-minute break
4. Repeat as needed

## Common Questions

### Can I customize the timer duration?

Yes you can! You can customize the timer duration in the settings.

### What happens when the timer ends?

You'll receive a notification and can choose to start a break or begin another session.

### Can I skip breaks?

Yes, but we recommend taking them to maintain productivity and prevent burnout.

## Tips for Success

1. Start with one 25-minute session
2. Use breaks to stretch and move
3. Keep your current task visible
4. Build up to more sessions gradually

> Remember: The Pomodoro timer is a tool to help you focus, not a strict rulebook. Adapt it to your working style.
